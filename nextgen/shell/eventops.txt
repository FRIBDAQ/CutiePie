Shell operations for event databases:

- evcreate - create a new event database file.
- evattach - attaches an event database to the current experiment database.
- evdetach - detaches an event database from the current experiment database.
- evinfo - Provides information about an event datbase file.
- evattinfo - Provides information about an attached database.
- evload    - Load events into a database from a CSV file.
- evaugment - Augment events in the event file.
- evuuid   - Get UUID of event database file.
